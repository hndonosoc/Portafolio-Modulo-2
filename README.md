Tienda de Productos de Madera – E-commerce

Este proyecto corresponde al Módulo 2 del portafolio, donde se desarrolló el frontend de una tienda e-commerce para un taller de madera artesanal.
La tienda muestra productos como:

-Galvanos
-Posavasos
-Llaveros
-Imanes souvenir
-Servilleteros
-Caja para bolsitas de té
-Marcos de cuadros
-Tablas de cocina

Objetivos

-Crear una tienda funcional usando tecnologías web
-Permitir navegar entre secciones
-Mostrar productos en tarjetas
-Incluir un carrito de compras
-Página de descripción individual del producto

Tecnologías utilizadas

-HTML5
-CSS3
-Bootstrap 5
-JavaScript
-LocalStorage
